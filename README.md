## Explicação do Projeto
 Este projeto simula um sistema de gerenciamento de empréstimo de livros em Java. O sistema permite o controle completo de livros, usuários e operações de empréstimo e devolução, com suporte a livros físicos e digitais.
 Funcionalidades:
1. Cadastro de Livros
Usando título, autor, ISBN e quantidade de cópias (apenas para livros físicos).

2. Cadastro de Usuários
Registro de nome e ID único. Além disso, cria um histórico de empréstimos vinculado ao usuário.

3. Empréstimo de Livro
Verificação de disponibilidade (para livros físicos) e registro do empréstimo no histórico do usuário.

4. Devolução de Livro
Atualização do acervo com a devolução e registro da data de devolução.

5. Listagem de Livros
Exibição de todos os livros cadastrados, indicação da quantidade de cópias disponíveis (para livros físicos).

6. Relatório de Empréstimos
Listagem dos livros com maior número de empréstimos em ordem decrescente. Além de apresentar o total de empréstimos realizados.

Pontos a melhorar:
Algumas funcionalidades desejadas, como um sistema para mostrar a lista de usuários e seus históricos, um sistema para monitorar os prazos de devolução e interfaces gráficas, não foram realizadas, pois não foram implementações listadas no PDF que contém o enunciado dos projetos.
